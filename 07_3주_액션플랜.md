- 수집 소스:
  - 유튜브: "AI generated video", "AI short film", "Sora", "Kling", "Runway" 검색
  - 틱톡: #AIvideo, #AIart, #AIfilm 태그
  - X(트위터): AI 영상 크리에이터 계정에서 직접 수집
- 카테고리 균형 배분 (시네마틱 20%, SF 15%, 밈/코미디 15%, 댄스 10%, 기타 40%)

### 시리즈 기획 (3개)
- 시리즈명, 컨셉, 에피소드 구성안, 대표 썸네일 컨셉 확정
- 각 시리즈의 첫 에피소드용 프롬프트 테스트 시작

### 프롬프트 테스트
- 06_프롬프트_세트.md의 프롬프트를 카테고리별로 실행
- 각 프롬프트당 2~3회 시도, 결과물 퀄리티 평가
- 잘 나오는 프롬프트 선별 → 2주차 양산에 활용

## 2주차 (4/14 ~ 4/20): 클립 양산 + 에피소드 제작

### 클립 양산 (목표: 50개)
- 1주차에 검증된 프롬프트로 집중 제작
- 하루 목표: 1인당 4~5개 = 2명 기준 하루 8~10개
- 5일 작업 시 50개 달성 가능
- 퀄리티 기준:
  - A등급 (10개): 후보정 + 사운드 → 홈 추천용
  - B등급 (25개): 기본 편집 + 배경음악 → 카테고리 피드용
  - C등급 (15개): AI 생성 그대로 + 프롬프트 상세 공개 → "나도 만들 수 있겠다" 느낌

### 에피소드 제작 (목표: 5개)
- 클립 3~4개를 엮어 하나의 에피소드로 구성
- 자막/나레이션 기반 스토리텔링 (대사 지양)
- 배경음악 필수 삽입
- 시리즈에 소속될 에피소드 우선 제작

### 라운지 추가 (나머지 30개)
- 1주차에 못 채운 분량 + 새로 발견한 영상 추가

### 2주차 일일 스케줄 (1인 기준)
| 시간 | 작업 |
|------|------|
| 오전 | 클립 제작 (프롬프트 실행 + 선별) 4~5개 |

## 3주차 (4/21 ~ 4/26): 시리즈 완성 + QA + 런칭 준비

### 시리즈 완성 (3~5개)
- 에피소드 추가 제작 5개 (시리즈당 3개 이상 구성)
- 시리즈 커버 이미지, 시놉시스, 메타데이터 완성
- 시리즈 내 에피소드 순서 배치

### 홈 큐레이션 배치
- "주간 인기" 섹션: 퀄리티 상위 콘텐츠 배치
- "추천" 섹션: 카테고리 다양성 확보하여 배치
- "인기 라운지": 반응 좋은 외부 영상 링크 우선 노출

### 최종 QA
- Playwright 스크린샷 캡처로 전 페이지 점검
- /clip 404 이슈 해결 확인
- og:image 설정 확인 (SNS 공유 프리뷰)
- 프롬프트/모델 정보 누락 여부 점검
- 저작권 연도 수정 (2024 → 2026)
- 404 에러 페이지 한국어화

### 런칭 준비
- X(트위터) 런칭 공지 작성
- 디시인사이드 AI갤 소개글 초안
- 카카오톡/네이버 공유 테스트
